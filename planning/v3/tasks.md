# Задачи v3 — статус реализации

1. Ролевая модель (глобальные + проектные роли, scoped admin, флаг согласования) — реализовано.
2. Субподряд по разделам WBS, назначение исполнителя из команды проекта, блокировка исполнителя после `approved` — реализовано.
3. Бухгалтер может менять статусы задач в «Мои задачи» — реализовано.
4. Структура организации: устранены дубли ролей и корректное отображение отсутствующего подразделения — реализовано.
5. Drag‑and‑drop в «Мои задачи» доступен всем глобальным ролям — реализовано.
6. ГИП в «Мои задачи» видит только назначенные ему задачи — реализовано.
7. В «Мои задачи» можно создать задачу с названием и сроком (личные задачи) — реализовано.
8. Личные задачи отображаются в таймшите как задачи вне проекта — реализовано.

Примечание: поле `completion_percent` физически осталось в БД, но в UI и логике не используется.
