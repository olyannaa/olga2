# План развития (по факту реализации)

## Сделано
- База и миграции (users, projects, tasks, WBS, time entries, approvals, contractors).
- Авторизация JWT, базовый RBAC.
- Проекты, архивирование/восстановление, команды проекта.
- WBS с авто‑задачами и ограничением глубины.
- Канбан по задачам, статусная история.
- Таймшиты (месяц/неделя), day‑off.
- Субподряды с согласованием.
- Аналитика по проекту.
- Чат проекта (REST).

## Частично
- Файлы проекта: есть API и метаданные, UI‑раздел пока заглушка.
- Настройки: экран доступен, изменения не сохраняются.
- Отчеты: есть сводка по time‑tracking, экспорт/Excel не реализован.

## В план (backlog)
- Реальная интеграция Nextcloud (папки, права, upload/download).
- Реалтайм‑чат/уведомления.
- Календарь/праздники (work_calendar) и полноценные табели.
- Расширенные отчеты и экспорт.
