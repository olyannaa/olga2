# Функциональные требования (актуальный статус)

## 3.1 Управление проектами

| Статус | ID | Название | Реализация |
| :---- | :---- | :---- | :---- |
| готово | ФТ-П-01 | Создание и управление проектами | CRUD, статусы active/completed/archived, архив/restore. |
| частично | ФТ-П-02 | Жизненный цикл проекта | Фиксированные статусы, без настраиваемого воркфлоу. |
| готово | ФТ-П-03 | WBS и диаграмма Ганта | WBS, ограничение глубины, план/факт Ганта. |
| частично | ФТ-П-04 | Базовый план | Шаблоны WBS локальные; фиксация базового плана не реализована. |

## 3.2 Управление задачами

| Статус | ID | Название | Реализация |
| :---- | :---- | :---- | :---- |
| готово | ФТ-З-01 | Создание и редактирование задач | Проектные/личные/бухгалтерские/субподряды, ограничения по WBS. |

| готово | ФТ-З-05 | Ролевой доступ к задачам | RBAC + проектные роли, ограничения по задачам бухгалтера. |
