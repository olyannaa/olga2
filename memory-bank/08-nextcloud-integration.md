# Интеграция с Nextcloud (текущее состояние)

## Что реализовано
- В БД есть таблица `project_folders`.
- Endpoint `POST /projects/:id/files/ensure-folder` создаёт запись со значением пути `/Projects/{projectId}` и статусом `stub`.
- Файлы проекта/задач хранятся как метаданные (имя + URL), без загрузки в хранилище.

## Что не реализовано
- Реальная интеграция с WebDAV/OCS API отсутствует.
- Автоматическое создание папок и выдача прав — только заглушка.
- UI раздел «Файлы» в проекте пока пустой (stub).

## Планируемая схема (после интеграции)
- Создание папки проекта в Nextcloud, хранение пути в `project_folders`.
- Связка доступа с участниками проекта.
- Хранение ссылок/etag/метаданных для файлов.
