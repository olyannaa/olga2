# Стек технологий

## Frontend
- React + Vite + TypeScript.
- Tailwind CSS + shadcn/ui.
- TanStack Query для работы с API и кеширования.
- React Hook Form + Zod используются в формах.

## Backend
- Node.js 18+ (типизация под Node 20).
- NestJS 10 (TypeScript).
- REST API (без WebSocket/SSE).

## База данных и миграции
- PostgreSQL 16.
- Миграции: node-pg-migrate.

## Аутентификация и безопасность
- JWT access/refresh.
- Хэширование паролей: Argon2.

## Файлы
- Хранение файлов не интегрировано; используется хранение метаданных/URL.
