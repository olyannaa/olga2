# Справочник пользователя

Этот справочник описывает фактическое поведение системы ISET Group и помогает работать в интерфейсе для каждой роли.

## Как пользоваться
- Найдите свой раздел по роли и модулю.
- Следуйте пошаговым инструкциям.
- В каждом модуле указаны ограничения и правила.
